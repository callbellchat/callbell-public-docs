# Se mettre en route

## Qu'est-ce que Zapier ?

Zapier est un outil qui vous permet de connecter les applications que vous utilisez tous les jours pour automatiser des tâches et gagner du temps. Vous pouvez vous connecter avec plus de 2 000 applications pour créer vos propres automatisations.

## Comment fonctionne Zapier ?

Zapier utilise un **Zap** pour connecter deux applications ou plus.

Un Zap est composé d'un **Déclencheur** et d'une ou plusieurs **Actions**.

Chaque fois que le déclencheur se produit dans une application, Zapier effectuera automatiquement les actions ou recherches dans une autre application.

## Comment utiliser Callbell avec Zapier

### Configurer votre Zap

Il existe deux façons d'intégrer Callbell avec d'autres applications : **Déclencheur** et **Action**.

![Zapier Déclencheur et Action](./assets/trigger+action.png)

### Déclencheur

Lorsqu'une nouvelle conversation est créée dans Callbell, vous pouvez envoyer les données vers une autre application. Par exemple, vous pouvez envoyer les données vers une feuille de calcul Google, un CRM ou un canal Slack.

### Action

Lorsqu'un événement se produit dans une autre application, vous pouvez envoyer les données vers Callbell. Par exemple, vous pouvez envoyer des données d'une feuille de calcul Google, d'un CRM ou d'un canal Slack vers Callbell.

### Exemple

Disons que vous souhaitez envoyer un message à un client lorsqu'une nouvelle commande est passée sur votre boutique en ligne. Vous pouvez utiliser Zapier pour envoyer les données de votre boutique en ligne vers Callbell.

Votre boutique en ligne est le **Déclencheur** et Callbell est l'**Action**.
