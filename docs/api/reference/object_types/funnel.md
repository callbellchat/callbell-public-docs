### Message Object

| Field       | Type   | Description                                        |
| :---------- | :----- | :------------------------------------------------- |
| `createdAt` | string | Date of conversation creation (ISO 8601 formatted) |
| `name`      | string | Name of the funnel                                 |
| `uuid`      | string | Unique identifier of the funnel                    |
| `emoji`     | string | Emoji icon for the funnel                          |
| `position`  | number | Position/order of the funnel stage                 |
